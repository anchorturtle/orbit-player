# 🪐 Orbit Planetary Player

> Cosmic High-Fidelity music player UI

A dark-mode, glassmorphism-styled music player built with plain HTML + Tailwind CSS CDN. No build step required.

## 🚀 Deploy to Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/anchorturtle/orbit-player)

1. Click the button above, or
2. In [Netlify](https://app.netlify.com), go to **Add new site → Import from Git**
3. Select this repo (`anchorturtle/orbit-player`)
4. Set **publish directory** to `/` (root)
5. Leave **build command** blank
6. Click **Deploy**

## 🛠 Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Google Fonts: Plus Jakarta Sans
- Material Symbols Outlined (icons)
